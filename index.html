Questions fréquentes

Tout ce que vous voulez savoir.

LegalPress génère des documents basés sur des prompts juridiques précis, rédigés selon les lois en vigueur. Cependant, ils ne remplacent pas une consultation juridique pour les situations complexes. Pour un site standard, ils offrent une conformité de base solide. Pour des activités réglementées (finance, santé, etc.), nous recommandons une validation par un avocat.
Oui, LegalPress utilise l'API Claude d'Anthropic pour générer les documents. Vous devez créer un compte sur console.anthropic.com et obtenir une clé API. Le coût par génération est très faible (quelques centimes) et la mise à jour automatique hebdomadaire reste économique.
Chaque semaine, le cron WordPress déclenche une régénération de tous vos documents actifs via l'API Claude. Les pages WordPress correspondantes sont mises à jour automatiquement. Vous recevez une notification si une modification significative est détectée.
Vos informations (nom de société, SIRET, adresse) sont stockées uniquement dans votre base de données WordPress. Elles sont transmises à l'API Claude uniquement lors de la génération des documents, et ne sont jamais conservées par Anthropic selon leur politique de données.
Oui, le plan Agency inclut la marque blanche : vous pouvez renommer le plugin avec votre propre branding et l'intégrer dans votre offre d'agence. Idéal pour proposer la conformité RGPD comme service additionnel à vos clients.
